## 2. お悩み共感セクション

### セクションタイトル
```
啓発広報担当者の、こんな声をよく聞きます
```

### 4つの悩み（カード形式）

#### 悩み1
```
「やっているのに、伝わってる手応えがない」
何度も啓発キャンペーンをしてきたが、本当に届いているのか、
変化があるのか、よくわからない。
```

#### 悩み2
```
「成果をどう測ればいいか分からない」
広報の効果が見えにくい。社内に説明する材料がなく、
次の予算確保にも苦労している。
```

#### 悩み3
```
「社内に説明できる言葉がない」
広報活動の意義は感じているが、定量的に語れない。
理事会・委員会で「やっていることの価値」を伝えきれない。
```

#### 悩み4
```
「届けたい層に、届かない」
特に若年層への啓発が機能していない。
SNS活用も検討したいが、何から始めればいいか分からない。
```

### レイアウト
```
■ セクション背景：薄いベージュ（#FAF8F2）
■ カード：白背景、左側にマスタード縦線（4px幅）
■ カードの並び：PC 2×2 / スマホ 縦積み
```

## 3. 提供サービス（3つの支援）

### セクションタイトル
```
私たちが提供する3つの支援
```

### リード文
```
啓発広報には、企画から制作、効果検証まで、
考えるべきことがたくさんあります。
私たちは、その全工程を伴走できる体制を整えています。
```

### 3つの支援（3カラム）

#### ① 伴走支援
```
■ キャッチ：啓発広報の、根っこから整理する。

■ 概要：
成果定義・KPI設計・効果検証の段階から、一緒に考えます。
「何を伝えるか」「誰に届けるか」を、対話を通じて
言語化することから始めます。

■ 提供内容：
・現状ヒアリング・課題整理
・成果定義・KPI設計
・効果検証の設計
・年間広報計画の策定

■ こんな方におすすめ：
・広報全体を俯瞰して整理したい
・「やっていることの意義」を社内に説明したい
・継続的なパートナーが欲しい
```

#### ② 企画・設計
```
■ キャッチ：届くための戦略を、組み立てる。

■ 概要：
ターゲット分析・アンケート調査・表現戦略・コンテンツ設計を行います。
「届かない理由」を構造的に分解し、戦略を組み立てます。

■ 提供内容：
・ターゲット分析・ペルソナ設計
・アンケート調査・インタビュー
・表現戦略・コンテンツ設計
・キャンペーン全体設計

■ こんな方におすすめ：
・特定のテーマで届かせたい層がある
・新しいキャンペーンを企画したい
・データに基づいた戦略を立てたい
```

#### ③ 制作・運用
```
■ キャッチ：伝わるかたちで、つくり、届ける。

■ 概要：
啓発動画（パラパラ漫画ほか）・印刷物・キャラクター制作・
SNS広告運用・効果測定レポートまで、一貫してご提供します。
「作って終わり」ではなく、「届くまで」を担います。

■ 提供内容：
・啓発動画制作（パラパラ漫画、アニメーションほか）
・印刷物制作（パンフレット、ポスターほか）
・キャラクター制作・ブランディング
・SNS広告運用・キャンペーン運用
・効果測定・レポート作成

■ こんな方におすすめ：
・具体的なコンテンツが必要
・SNS運用までまるごと任せたい
・効果測定もセットで頼みたい
```

### レイアウト
```
■ 3カラムの均等配置
■ 各カラム：
　・上部にネイビー帯
　・丸数字バッジ（①②③）
　・支援名（大、明朝）
　・キャッチ（中、明朝・斜体）
　・概要本文（小、ゴシック）
　・提供内容リスト
　・「こんな方におすすめ」
```

## 5. パラパラ漫画が効く4つの理由

### セクションタイトル
```
啓発広報に、パラパラ漫画が効く4つの理由
```

### 4つの理由（4カラム）

#### 理由1：余白で読み手を引き込む
```
パラパラ漫画には、ナレーションやBGMの「余白」があります。
情報を一方的に押し付けず、視聴者自身が想像し、
気持ちを乗せながら読み進められます。
```

#### 理由2：重い話を、柔らかく
```
法律・制度・義務に紐づく啓発テーマは、どうしても堅くなりがち。
手描きの温度感が、その堅さを和らげ、
「自分にも関係ある話」として受け取ってもらえます。
```

#### 理由3：最後まで見たくなる
```
ストーリー構造を持たせることで、視聴完了率が上がります。
家電製品協会様の事例では、一般的な動画広告の3倍の
視聴完了率を実現しました。
```

#### 理由4：複数媒体に展開できる
```
1つの作品から、動画版・印刷版・SNS用の短尺版など、
複数の媒体に展開可能です。
コンテンツ資産として、長く活用できます。
```

### CTAボタン
```
パラパラ漫画の詳細・制作実績を見る → （→ parapara-manga.com）
```

### レイアウト
```
■ 4カラム配置
■ 各カードに小さな手描きイラスト or アイコン
■ スマホ：縦積み（または2×2）
```
