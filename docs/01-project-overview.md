# 01. プロジェクト概要

## サイトの位置づけ

「**啓発広報ラボ**」は、株式会社アトムストーリーが運営する、業界団体・公益法人・一般社団法人の啓発広報を支援する専門メディア兼サービスサイトです。

### 主な役割

```
1. アウトバウンド営業先（業界団体・公益法人の広報担当者）の信頼性担保
2. 商談化率の向上（営業先がサイトを見て「ちゃんとした専門会社」と判断する）
3. SEO/GEO対策によるプル型集客への転換
```

---

## ターゲット

### メインターゲット
業界団体・公益法人・一般社団法人の広報担当者・事務局長

### サブターゲット
- 行政の広報担当者
- NPOの担当者
- 啓発広報の発注を検討している意思決定者

### ターゲットの特徴
- 業務中にPCで、あるいはメール経由でスマホでサイトを開く
- 真面目で堅実なプロフェッショナル層
- 派手なデザインや軽い表現は信頼を損なう

---

## 3サイト構成

アトムストーリーが運営する3つのサイト：

| サイト | URL | バーティカル | 主なキーワード |
|---|---|---|---|
| アトムストーリー | atom-story.com | 会社の本体 | 会社名・概念名 |
| パラパラ漫画制作ラボ | parapara-manga.com | 手法特化 | 「パラパラ漫画 制作」など |
| **啓発広報ラボ** | **keihatsu-koho-lab.jp** | **用途特化** | **「啓発広報」「業界団体」など** |

3サイトは強い相互リンクで結ばれ、SEO評価を相互強化します。

## 実装の優先順位

```
Phase 1（最優先）：基本ページの実装
　1. トップページ（/）
　2. サービスページ（/service/）
　3. 事例詳細ページ（/case/kaden-seihin-kyokai/）
　4. お問い合わせフォーム（/contact/）

Phase 2：追加ページ
　5. 会社概要（/about/）
　6. 代表メッセージ（/message/）
　7. ナレッジ記事一覧（/knowledge/）
　8. ナレッジ記事詳細（/knowledge/[slug]/、microCMS連携）
　9. インタビュー記事（/interview/）

Phase 3：SEO/GEO対策
　10. 構造化データ全7種類
　11. メタ情報の個別設定
　12. サイトマップ（sitemap.xml）生成
　13. robots.txt
　14. Googleアナリティクス・Search Console連携
```

### URL命名ルール
- すべて末尾スラッシュ
- 英語スラッグはハイフン区切り
- 日本語URLは使わない
- 将来追加するページも同じディレクトリ構造に揃える
